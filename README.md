# Bank-Management-System-Python-OOP
A simple Bank Management System built in Python using Object-Oriented Programming and file handling. It allows users to create accounts, deposit and withdraw money, and view transaction history. Developed as a learning project to strengthen core programming, problem-solving, and software design skills.

## Features
- Create new bank accounts
- Deposit and withdraw money
- View account details
- Transaction history stored in a file
- Persistent account data using file handling

- ## 🛠 Tech Stack
- Python  
- Object-Oriented Programming (OOP)  
- File Handling  

## 📂 Project Files
- `Bank-Management-System.py` → Main program file  
- `accounts.txt` → Stores account details  
- `transactions.txt` → Stores transaction history
